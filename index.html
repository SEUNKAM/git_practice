HTML Table Reference

Table Tags and Atributes

Tag Name Description
<table> Table A table is rows and columns arrangements of data
<th> Table Head Defines a header cell in a table
<tr> Table Row/ Defines a row in the table
<td> Table Data Defines a cell in the table
<caption> Caption Defines a table caption
<colgroup> Columns Group Specifies a group of one or more columns in a table for formatting
<col> Column Specifies column properties for each column within a <colgroup> element
<thead> Table Head Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Foot Groups the footer content in a table
rowspan Row Span The value of the rowspan attribute represents the number of rows to span
colspan Column Span The value of the colspan attribute represents the number of columns to span
